# Comic Filters
Several comic filters implemented with Dart.

### Current Filters
- Detail Enhancement Filter
- Pop-art Filter
- Pencil Edges Filter
- Bilateral Filter
- Pencil Sketch Filter
- Pixelate Filter
- Quantization Filter
